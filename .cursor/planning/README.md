Carpeta `planning/` con la documentación completa para el desarrollo del MVP y la evolución a plataforma SaaS.

- `00-Executive_Summary.md` — Resumen ejecutivo y objetivos.
- `01-Vision_General.md` — Visión, público y scope.
- `02-PRD.md` — Requisitos funcionales y no funcionales.
- `03-Architecture.md` — Arquitectura técnica y decisiones.
- `04-Data_Model.md` — Modelo de datos relacional.
- `05-API_Spec.md` — Contrato API (visión y endpoints críticos).
- `06-Roadmap.md` — Roadmap por fases y hitos.
- `sprints/` — Carpetas `sprint-{tema}` con informes por sprint y estimaciones.
- `07-Design_System.md` — Tokens y componentes (Next.js + Tailwind).
- `08-QA_and_Testing.md` — Plan de pruebas y criterios de salida.
- `09-Security_Privacy.md` — Controles de seguridad y GDPR.
- `10-Deployment_Runbook.md` — Runbook de despliegue y rollback.
- `11-Monitoring_Observability.md` — Métricas, alertas y dashboards.
- `12-GoToMarket.md` — Estrategia de lanzamiento y marketing MVP.
- `13-Budget_and_Estimations.md` — Estimaciones de coste y recursos.

Instrucciones: revisa los sprints en `sprints/` para ver las tareas desglosadas con estimaciones por sprint.
